## Installation

You can install the package via composer:

```bash
composer require toolstud-io/nowtricity
```

## Usage

``` php
$obj = new ToolstudIo\Nowtricity();
echo $obj->echoPhrase('Hello, toolstud-io!');
```

## Testing

